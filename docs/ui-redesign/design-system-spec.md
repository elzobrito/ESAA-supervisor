# UIR-SPEC-006: Design System

## Objetivo
Definir a linguagem visual do redesign para dar unidade, sobriedade e hierarquia ao ESAA Supervisor.

## Direcao Visual
- dashboard profissional e tecnico
- contraste claro entre estrutura, dados e estados operacionais
- menos ruído visual e mais agrupamento semantico

## Tipografia
- hierarquia forte para titulos e KPIs
- texto funcional legivel para tabelas e paineis densos
- labels de apoio menores, sem ficar apagados

## Espacamento
- escala previsivel e modular
- cards, tabelas e paineis devem compartilhar ritmos visuais
- espacamento reduzido em superficies densas, mais amplo em landing executiva

## Componentes Base
- cards
- badges
- tabelas
- filtros
- drawers
- modais
- toolbars
- KPI cards

## Estados Semanticos
- sucesso
- alerta
- erro
- info
- neutro

## Regras de Hierarquia
- informacao critica primeiro
- acentos de cor apenas para significado
- containers devem competir menos entre si

## Resultado Esperado
- o produto precisa parecer um cockpit operacional sofisticado, nao uma pagina de blocos desconexos
