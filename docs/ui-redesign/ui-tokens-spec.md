# UIR-SPEC-006: UI Tokens

## Tipografia
- `font.family.base`
- `font.family.display`
- `font.size.xs/sm/md/lg/xl/2xl`
- `font.weight.regular/medium/semibold/bold`

## Espacamento
- `space.2`
- `space.4`
- `space.8`
- `space.12`
- `space.16`
- `space.24`
- `space.32`

## Radius
- `radius.sm`
- `radius.md`
- `radius.lg`
- `radius.pill`

## Cores Base
- `color.bg.canvas`
- `color.bg.panel`
- `color.bg.elevated`
- `color.fg.primary`
- `color.fg.muted`
- `color.border.subtle`

## Estados
- `color.state.success`
- `color.state.warning`
- `color.state.error`
- `color.state.info`

## Tokens de Componente
- `card.padding`
- `card.shadow`
- `badge.height`
- `table.row.height`
- `drawer.width.desktop`
- `modal.max_width`

## Regras
- badges, cards, tabelas e painéis devem herdar tokens compartilhados
- nenhum componente central deve usar valores arbitrarios fora do sistema
- os tokens devem suportar sidebar, topbar, cards KPI, overlays e grids densos
